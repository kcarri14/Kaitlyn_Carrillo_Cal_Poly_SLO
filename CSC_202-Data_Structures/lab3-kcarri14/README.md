# Lab 3: Analysis of Algorithms

## Overview
In this lab, you will explore and analyze the efficiency of various algorithms. Through practical implementation and comparison, you will gain a deeper understanding of asymptotic notation, algorithm optimization, and the trade-offs between different algorithmic approaches.

## Tasks

### Task 1: Linear Search vs. Sort + Binary Search

**Objective:** Compare the efficiency of linear search versus a combination of sorting followed by binary search.

**Instructions:**
1. You will be given several long sorted lists of integers.
2. Implement a linear search algorithm to find a specific element in each list.
3. Implement a sorting algorithm followed by a binary search to find the same element.
4. Record the number of comparisons made by each method.
5. Graph the number of comparisons against the list size (n) for both methods.
6. Return a list of the differences. 
7. Submit your plotted image
8. Answer the best-worst case questions

### Task 2: Timing Sorting Algorithms

**Objective:** Compare the execution time of selection sort and insertion sort.

**Instructions:**
1. Several lists will be provided to you.
2. Implement both selection sort and insertion sort algorithms.
3. Measure the execution time for sorting each list using Python's `timeit` module.
4. Plot the execution times against the list size (n) for both sorting algorithms.
5. Return the list of execution times


## Assessment
Your submission will be evaluated based on:
- Correctness and efficiency of the implemented algorithms, verified through comprehensive test cases.
- Accuracy and clarity of the graphs and analysis.
